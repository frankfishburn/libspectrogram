# libspectrogram
## About
**libspectrogram** is a C++ library for computing spectrograms and periodograms, with a C interface and MATLAB bindings

---
## Building
### Library
```bash
mkdir build
cd build
cmake ..
make
make test
```
### Matlab
```Matlab
cd('bindings/matlab');
make;
```
